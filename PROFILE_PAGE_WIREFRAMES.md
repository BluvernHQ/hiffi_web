# Profile Page - Mobile Responsive Wireframe Documentation

This document provides detailed wireframe documentation for the mobile responsive profile page (`/profile/[username]`).

## üì± **Breakpoints**

| Breakpoint | Min Width | Device Type |
|------------|-----------|-------------|
| **xs** | 0px | Mobile phones (portrait) |
| **sm** | 640px | Mobile phones (landscape), small tablets |
| **md** | 768px | Tablets (portrait) |
| **lg** | 1024px | Tablets (landscape), small laptops |
| **xl** | 1280px | Desktops, large laptops |

---

## üéØ **Page Variations**

The profile page has **two main layouts**:

1. **Personal Profile View** - Regular user viewing their own profile
2. **Standard Profile View** - Creator/admin profile or viewing someone else's profile
